# PHAEducation
This is an escape room game to learn pharmacology. 
At the end of the game, the learners should be able to:
    1.    Define and differentiate agonist and antagonist
    2.    Match a target protein to an appropriate drug and therapeutic indication
    3.    Explain the mechanism of action (MoA) at the target protein in their own words
    4.    Predict the key adverse effects from the target and drug class
    5.    Compare the efficacy of the drug for the same disease category
    6.    Comprehend the strategies of drug design
